# p10k

> Manage configurations for powerlevel10k.
> More information: <https://github.com/romkatv/powerlevel10k>.

- Configure powerlevel10k interactively:

`p10k configure`

- Reload powerlevel10k:

`p10k reload`

- Display help:

`p10k help`
