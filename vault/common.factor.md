# factor

> Prints the prime factorization of a number.
> More information: <https://www.gnu.org/software/coreutils/factor>.

- Display the prime-factorization of a number:

`factor {{number}}`

- Take the input from stdin if no argument is specified:

`echo {{number}} | factor`
